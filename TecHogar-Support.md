🖥️ TecHogar Support Perú

Plataforma web de soporte técnico a domicilio en Lima Metropolitana.

📋 Descripción
Sitio web desarrollado como Proyecto Final 

🚀 Tecnologías Utilizadas
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript ES6+ (Vanilla JS)
- Font Awesome 6.5
- Google Fonts (Poppins)

 ✅ Funcionalidades
- Menú responsivo con botón hamburguesa
- Formulario con validación en JavaScript
- Ventana modal informativa
- Selector de distrito con cobertura dinámica
- Almacenamiento con localStorage
- Página de confirmación de reserva
- Integración con WhatsApp
